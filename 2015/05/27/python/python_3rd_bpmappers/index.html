发表于 2015-05-27   |   分类于 python   |  

简介

bpmappers库是将python对象转化为字典类型的库, 在对外提供数据服务类型的功能开发用途较大，尤其后台使用django开发，bpmappers库简便将对象转化为dict类型，配和json.dumps功能对外提供json格式数据

安装

pip install bpmappers
